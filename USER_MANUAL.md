# SecZim Games Predictor Manuals

There are separate manuals for the two main audiences.

## Teams

Use the team guide if you need to:

- View standings.
- Log in as a team.
- Make predictions.
- Save predictions.
- Change your team PIN.
- Understand team points.

Open [TEAM_MANUAL.md](TEAM_MANUAL.md).

## Administrators

Use the admin guide if you need to:

- Settle results.
- Add games.
- Register teams.
- Reset team PINs.
- Award actual points.
- Delete incorrect point awards.
- Manage admin-only tasks.

Open [ADMIN_MANUAL.md](ADMIN_MANUAL.md).

Admin passwords should be shared privately, not written into these manuals.
